Thông tin máy chủ

Kết nối NovaMC nhanh và chính xác

Chọn đúng phiên bản bạn đang chơi, sao chép địa chỉ tương ứng rồi làm theo hướng dẫn bên dưới. Java dùng IP chính, Bedrock/MCPE dùng cùng IP và thêm port.

Server sẵn sàng
01Hướng dẫn Java Edition
  1. Mở Minecraft Java Edition phiên bản 1.21.5 trở lên.
  2. Chọn Multiplayer → Add Server.
  3. Nhập Server Address: play.novamc.asia.
  4. Bấm Done, chọn NovaMC rồi bấm Join Server.
02Hướng dẫn Bedrock / MCPE
  1. Mở Minecraft Bedrock hoặc Minecraft PE.
  2. Vào Play → tab Servers → Add Server.
  3. Nhập Address: play.novamc.asia.
  4. Nhập Port: 25552, lưu lại rồi kết nối.
💡
Nếu không kết nối được

Hãy kiểm tra đúng phiên bản, nhập đúng port cho Bedrock/MCPE, thử IP phụ novamc.usga.me:25552 hoặc liên hệ Discord NovaMC để được hỗ trợ.
